# 평가 기준 (Evaluation Criteria)

## 포지션: AX(AI Transformation) Engineer

### 직무 설명
AI 기반 개발 도구(특히 Claude Code)를 활용해 빠른 프로토타이핑과 솔루션 구축을 담당합니다.
클라이언트의 비즈니스 문제를 AI 도구로 해결하는 AX 컨설팅 역할입니다.

### 평가 항목 (Criteria)

각 항목은 1-10점으로 평가합니다.

#### 1. Claude Code 활용 능력 (claudeCodeProficiency)
- **가중치:** 높음
- **우수:** Claude Code로 실제 프로젝트를 빌드한 경험, multi-agent 활용, 커스텀 스킬 제작
- **보통:** AI 코딩 도구 사용 경험 (Copilot, Cursor 등), Claude Code는 미경험
- **미달:** AI 도구 사용 경험 없음

#### 2. 빠른 프로토타이핑 (rapidPrototyping)
- **가중치:** 높음
- **우수:** 짧은 시간 내 PoC/MVP 다수 제작 경험, 스타트업 환경
- **보통:** 프로젝트 경험 있으나 속도보다 완성도 중심
- **미달:** 프로토타이핑 경험 부족

#### 3. 지원 동기 (motivation)
- **가중치:** 중간
- **Tally 질문:** "왜 함께 일하고 싶나요?"
- **우수:** Joshua의 비전에 공감, AX 분야에 대한 구체적 관심, 성장 욕구 명확
- **보통:** 일반적인 AI 관심사, 구체성 부족
- **미달:** 동기 불분명 또는 포지션과 무관한 답변

#### 4. 일과 비전 (dailyWorkVision)
- **가중치:** 중간
- **Tally 질문:** "어떤 업무와 일과로 하루를 보낼 것 같으신가요?"
- **우수:** 현실적인 일과 + PoC 사이클 + 클라이언트 소통 언급
- **보통:** 개발 중심 일과, 컨설팅 측면 부족
- **미달:** 비현실적이거나 관련 없는 답변

#### 5. 기술 스택 적합성 (techStackFit)
- **가중치:** 보통
- **우수:** Supabase, SvelteKit, Cloudflare Workers 실무 경험
- **보통:** 유사 기술 경험 (Firebase, React/Next.js, Vercel)
- **미달:** 웹 개발 경험 부족

### 점수 기준 (Verdict Thresholds)

- **strong** (강력 추천): 총점 80 이상
- **maybe** (검토 필요): 총점 55-79
- **pass** (불합격): 총점 55 미만

### 총점 계산

가중 평균 방식:
- claudeCodeProficiency: 25%
- rapidPrototyping: 25%
- motivation: 20%
- dailyWorkVision: 15%
- techStackFit: 15%

### 레드 플래그 (Dealbreakers)

- 답변이 모두 1-2줄로 성의 없음
- 이력서가 포지션과 완전히 무관
- 복붙으로 보이는 답변
